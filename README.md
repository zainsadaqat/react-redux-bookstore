# Redux BookStore

## Built With

- HTML, CSS
- React, Redux

## Steps to run the project in your local machine

Go To Terminal and paste the following command:

`git clone git@github.com:zainsadaqat/react-redux-bookstore.git`

`cd react-redux-bookstore`

`npm install`

`npm start`
